\# evidence-timeline-builder



A minimal tool to reconstruct a chronological timeline from an evidence

`manifest.json` file.



Designed to work with evidence bundles generated by

\[`incident-evidence-packager`](https://github.com/maryamoah/incident-evidence-packager).



\## What it does

\- Reads `manifest.json`

\- Orders evidence files by collection time

\- Produces:

&nbsp; - a human-readable Markdown timeline

&nbsp; - a CSV timeline for analysis or reporting



\## Usage



```bash

python timeline\_builder.py <manifest.json>
